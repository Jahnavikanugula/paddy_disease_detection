# Paddy Disease Detection System

## 📌 Project Overview
This project detects diseases in paddy leaves using Machine Learning. It helps farmers identify diseases early and take proper action.

## 🎯 Objective
To classify paddy leaf images into different disease categories.

## 🛠 Technologies Used
- Python
- NumPy, Pandas
- TensorFlow / Keras
- OpenCV
- Matplotlib

## 📊 Dataset
The dataset contains images of paddy leaves with multiple disease classes such as:
- Bacterial Leaf Blight
- Brown Spot
- Blast
- Healthy Leaf
